bootstrap是前端框架，css样式框架
包含bootstrap.min.css,bootstrap.min.js两个文件，但在引入后面js文件时，需先引入jQuery文件
<link rel="stylesheet" href="../bootstrap/bootstrap.min.css">
<script src="../jquery-3.3.1.min.js"></script>
<script src="../bootstrap/bootstrap.min.js"></script> 
里面主要定义了类的样式，所以需要在HTML中增加该类，从来可以使用相应的样式
css,布局组件，插件。

1.垂直表单（默认）
2.内联表单	：它的所有元素是内联的，向左对齐的，标签是并排的
3.水平表单	：水平布局
创建垂直/基本表单的步骤：
	1.向父 <form> 元素添加 role="form"。
	2.把标签和控件放在一个带有 class .form-group 的 <div> 中。这是获取最佳间距所必需的。
	3.向所有的文本元素 <input>、<textarea> 和 <select> 添加 class ="form-control" 。
创建内联表单的步骤：
	在基本表单的基础上，向 <form> 标签添加 class .form-inline即可。
	使用 class .sr-only，您可以隐藏内联表单的标签
创建水平表单的步骤：(在基本表单的基础上)
	1.向父 <form> 元素添加 class .form-horizontal。
	2.把标签和控件放在一个带有 class .form-group 的 <div> 中。
	3.向标签添加 class .control-label。

所有的插件依赖于 jQuery。所以必须在插件文件之前引用 jQuery
使用bootrap插件：
1.data属性
2.JavaScript方法

Bootstrap 模态框（Modal）插件：
.modal					：定义一个模态框
.fade					：淡入淡出
.modal-dialog			：在.modal-content前定义div，里面添加该类
.modal-content			：模态窗口的内容，包括头部，身体，底部
.modal-header			：模态窗口的头部，包含标题
.modal-title			：模态窗口的标题，在.modal-header里包含
.modal-body				：模态窗口的身体
.modal-footer			：模态窗口的底部
data-backdrop			：指定一个静态的背景，当用户点击模态框外部时不会关闭模态框
data-keyboard			：当按下 escape 键时关闭模态框，设置为 false 时则按键无效
data-show				：当初始化时显示模态框
data-remote
.modal(options)			：js方法
.modal('toggle')
.modal('show')
.modal('hide')
show.bs.modal			：事件
shown.bs.modal
hide.bs.modal
hidden.bs.modal

data-spy="scroll"			：创建一个滚动监听事件

$('#myScrollspy').on('activate.bs.scrollspy', function () {
  // 执行一些动作...

Bootstrap 标签页（Tab）插件：
以不同的方式来激活各个标签页：(内容动态显示)
data-toggle="tab"							：标签页插件的使用
show.bs.tab									：在新标签页被显示之前触发
shown.bs.tab								：在新标签页被显示之后触发
$('#myTab a[href="#profile"]').tab('show')	：通过名称选取标签页
$('#myTab a:first').tab('show')				：选取第一个标签页
$('#myTab a:last').tab('show')				：选取最后一个标签页
$('#myTab li:eq(2) a').tab('show')			：选取第三个标签页（从 0 开始索引）
$(function(){
	$('a[data-toggle="tab"]').on('shown.bs.tab', function (e) {
	// 获取已激活的标签页的名称
	var activeTab = $(e.target).text();
	// 获取前一个激活的标签页的名称
	var previousTab = $(e.relatedTarget).text();
	$(".active-tab span").html(activeTab);
	$(".previous-tab span").html(previousTab);
	});
});

Bootstrap 警告框（Alert）插件：
data-dismiss="alert"
.alert()				：让所有的警告框都带有关闭功能。
.alert('close')			：关闭所有的警告框(需要给每个警告框都加上)
close.bs.alert
closed.bs.alert
$('#myalert').bind('close.bs.alert', function () {
  // 执行一些动作...
})
可以向所有的警告框消息添加可取消（dismiss）功能，有以下两种方式启用警告框的可取消（dismissal）功能：
<a class="close" data-dismiss="alert" href="#" aria-hidden="true">
    &times;
</a>
$(".alert").alert()

Bootstrap 按钮（Button）插件：
data-loading-text="Loading..."	：加载状态
data-toggle="button"			：切换，可用于单个，checkbox,radio
$().button('toggle')			：切换按压状态
$().button('loading')			：当加载时，按钮是禁用的
$().button('reset')				：重置按钮状态，文本内容恢复为最初的内容
$().button(string)				：与data-complete-text="Loading finished"一起使用，$(this).button('complete');

Bootstrap 折叠（Collapse）插件：
data-toggle="collapse"		：表示是一个折叠组件
href/data-target			：值是子组件的id#
data-parent					：值是整体的id，整体用来控制所有折叠面板#
.collapse					：隐藏
.collapse.in				：显示
.collapsing					：当过渡效果开始时被添加，当过渡效果完成时被移除
.collapse(options)			：激活内容为可折叠元素
.collapse('toggle')			：切换显示/隐藏可折叠元素
.collapse('show')			：显示可折叠元素
.collapse('hide')			：隐藏可折叠元素
show.bs.collapse
shown.bs.collapse
hide.bs.collapse
hidden.bs.collapse
$('#identifier').collapse('toggle')

Bootstrap 轮播（Carousel）插件：
data-slide				：接受关键字 prev 或 next
data-slide-to			：向轮播传递一个原始滑动索引
data-ride="carousel"	：标记轮播在页面加载时就开始动画播放
data-interval			：自动循环每个项目之间延迟的时间量//number
data-pause				：鼠标进入时暂停轮播循环，鼠标离开时恢复轮播循环//hover
data-wrap				：轮播是否连续循环//true/false
.carousel(options)		：初始化轮播为可选的 options 对象，并开始循环项目
.carousel('cycle')		：从左到右循环轮播项目
.carousel('pause')		：停止轮播循环项目
.carousel(number)		：循环轮播到某个特定的帧
.carousel('prev')		：循环轮播到上一个项目
.carousel('next')		：循环轮播到下一个项目
slide.bs.carousel		：当调用 slide 实例方法时立即触发该事件
slid.bs.carousel		：当轮播完成幻灯片过渡效果时触发该事件
